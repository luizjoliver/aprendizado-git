Este repositório é referente a parte de git e github do Curso full Stack One Bit code , inicalmente na introdução foi explicado a diferença entre git e github. Enquanto git é um software que faz controle de versão na sua máquina , o github é um repositório em nuvem que pode ser conectado juntamente com git de uma máquina , inclusive isto que faz o git e github ser tão eficiente , pois eles ajudam bastante a produção de projetos em equipe , de forma segura e organizada.
